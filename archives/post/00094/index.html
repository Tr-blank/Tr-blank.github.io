【記事】累癱的 大一迎新 2015-05-03

( 累累累累累 )

10/9 、 10/10 是資管＋應日學長姐們規劃籌辦的迎新。

是我跟過活動最多的團，也是玩的最開心的一次。其實原本還有點猶豫要不要參加，因為對同學們都還不熟就要來個兩天一夜的活動，實在讓我有些卻步，我算是個會怕生的人。不過很慶幸我最後還是參加了，如聽說的一樣，大一迎新真的會是大學生活中難忘的回憶之一。

10/9 星期六早上 6:30 就要集合，當初聽到這時間時，真覺得好早好早，雖然我就住在學校宿舍裡，但我是晚睡晚起派的人，真的超怕會睡過頭。用手機在 5:00~5:50 設了五個鬧鐘，當天早上也跟預期的一樣，每次手機一震就按掉躺回床上，直到快 6:00 才克制住繼續賴床的慾望，不然我想，再給他躺一次的話大概就會錯過這次的活動了。

因為快 6:00 才起床，到達集合地點時就差不多 6:30 ，來不及去買早餐，所以在遊覽車上心裡不時念著好餓好餓。不過在車上時有玩一些小遊戲和互相認識小隊成員和隊補的時間，想著肚子餓的心思就被分散了。

在這從學校做車到目的地的時間裡，覺得隊補們把時間控制得很好，沒有以往跟團那種不知道要做啥只好望窗外發呆的情況，也在這段時間裡，初步認識了很多人。

我被分到的小隊是第三組 “ 小象邦邦隊 ” ，加隊補的話有 11 人，其中女隊補和四位女隊員是應日的，資管的就我和一位女同學、三位男同學＋男隊補。

到達目地豐原的流星花園使業式完後，就開始各組分散練習隊呼和隊嗆。各組的隊補們已經事先想好，免去以往跟團大家都沒念頭的沉默狀態。那時就感受到，學長姐們一定是準備很久了，超用心的。

我們的隊呼是：

　　第三小隊在哪裡？

　　第三小隊在這裡。

　　上山下海沒問題，

　　樣樣比賽得第一。

　　我們是，第三小隊！！

第一次聽到 “ 隊嗆 ” 這詞，感覺超新鮮。藉由隊嗆，很大聲的喊出髒話也不會覺得有違和感 XDDD( 喂 !

我們的隊嗆：

　　跨三小！跨三小！

　　三小隊，在這裡。

　　遇到我們滾邊去，

　　趕快回家吃自己！

（這些還有搭配動作，只是我不太會用說的 -..-.. 所以就省略了（喂）

這次迎新還有個特別的小遊戲穿插在這兩天的各個活動間，叫做 “ 小主人 & 小僕人 ” 。

在紙上寫上所有參加這次迎新的學生，然後讓每個人抽，抽到的那位就是自己的小僕人，可以在紙上寫要指定小僕人做的事，指定卡會全部收回學長姐那邊，在吃完飯後或是一些活動的串場讓小僕人上台做出指定卡上寫的事情。

參加的學生很多，所以不太可能每個僕人都會被叫上去做，好像學長姐們有先看過指令卡，挑出幾張適合的，所以每到了這遊戲時間，都很有笑點，但又不會太超過。

在迎新開始前幾天，學長有提醒我們要準備 50 元左右的小禮物。

學校附近就有家墊腳石，就打算在那買一買，那天在墊腳石的禮品區看到好多同班同學都在挑禮物，因為不知道是要送誰，所以大家都很傷腦筋要送什麼禮物，逛了很久。我最後還是回到文具區，還是比較想以實用為主。

本人不太愛裝飾品，所以也不想送那類的東西。

買了一個小錢包、一本小筆記本、一之三色筆，因為都在特價所以就湊一湊到 50 多了，一個很平凡的禮物。禮物的包裝讓我很苦手，亂折亂黏，包出一個像是便利商店裡賣的去邊夾心吐司形狀的禮物。

在迎新那天，我水壺漏水，小禮物不幸遭殃，變的破破爛爛。每看到一次小禮物，心裡就自嘲一次，為自己的臉皮多加一層。

而那個小禮物在最後就是要送給自己抽到的小僕人。在經歷過 N 次的緊張抽指令卡遊戲後，終於可以知道自己的小主人是誰，然後接受他送的禮物。

我收到的是 “ 舒適氣枕 ”… 很特殊的禮物，也算是實用型拉，比收到裝飾品來的讓我開心多了，感謝小主人和慶幸我沒被抽上台做指令 ( 雖然我不知道是啥指令 ) 。

第一天下午有闖關遊戲，一共有五關。都很有趣，但我印象最深刻的是前往各個關卡接近 60 度的瘦身坡，我看大部分的體力（脂肪）都耗在那邊了。

而且有幾關真的是運氣問題，上天無情，除了無奈還是只有無奈，最後我們第九名（倒數第二）。

雖然成績不是很好，但我卻感覺我們小隊反而更團結了，或許是經過這活動彼此又多認識一點的關係。

而 晚餐讓我有點意外的是烤 肉，因為要為一大群的人準備烤肉的用具和要找尋合適地點是件很麻煩 ( 困難 ) 的事，而且又要花很多錢買食材道具 ( 應該比請餐廳準備來的貴吧 ) 。雖然食材不算很好，但我們都吃得很飽，也烤得很盡興。讓我又更加佩服學長姐的能力了。 　

吃完晚餐再來就是晚會了，學長姐的表演很賣力、很精彩，感受到他們是真心歡迎我們加入這學校、這科系。

晚會最後都會來個男女合跳，這次是男生在外圍一圈，女生在內為一圈，一起跳幾段舞步後女生就往旁邊轉換舞伴。雖然知道這樣已經算保守了，但還是會有點尷尬。

在跳這舞時，請原諒我不敢抬頭，雖然知道這樣似乎不太禮貌，但我還是不太想抬頭。

因晚會是在室內舉行（有開冷氣），所以我的手就會比較低溫，隨著音樂的飄揚，轉過了幾位舞伴，手的溫度也漸漸從低溫回歸到常溫，在不敢抬頭的情況下，注意力就會被牽住的手和跳的舞步拉過去。

透過被牽住的手和每個人舞步的些許差異，似乎可以理解一些事情，握的力道、位置、手心的觸感、溫度、濕度、舞步的節奏、流順感 …… 或許我尷尬的心情也傳過去對方那了吧。

晚會再來就是夜遊！！

說起對夜遊的回憶，都是一大群人（幾十個）浩浩蕩蕩的出發，而我都是會走在隊伍的中後方，有扮鬼嚇人也都是嚇在前方的女生群，在中後方的我也只會聽到前方女生們的尖叫，然後完全看不到前方發生啥事。

但是！這次完全不一樣了拉！！！

一組就 11 人，整個夜遊路線只會有一組在裡頭，路線是在一條旁邊是果園（橘子樹）還有草叢的小徑，而且路上也有一些音樂、冥紙、血字 …… 討厭的氣氛！！嚇人的學長身上也會有些血跡的打扮，然後都是在離我們很接近的地方從旁邊草叢下方竄出上半身嚇人＋大吼。

每次出現我都被嚇到（掩面），雖然我不會像之前女生那樣尖叫，但我的是反應動作很大，一出現就馬上反射的往另一邊退（一大步到 N 大步），導致被我抓著手的女同學，老是被我拉走，而我通常都是往後退，後面就是男生群，他們似乎也被我撞了不少次 ……

有時學長嚇完後沒有馬上回到草叢裡，而是在原地維持原本動作小幅度的擺盪，而我當時雖然知道他們是人不是鬼 …… 但也還是僵硬的在旁邊警戒著盯著那 ” 鬼 ”……( 丟臉丟到爆炸，形象毀壞了（嘆 ) 。

我想我除了怕鬼，也還是有點怕人吧。

晚上我們住的是小木屋，小小一間四人房，把下鋪的床拉出來就幾乎把房間全占滿了，但比起其他系的搭帳篷要來的好很多了。

第二天早上 6:50 就要集合做早操，又是個考驗自制力的早晨。

在跳完早操，值星官在要來帶我們回去吃早餐時，出了點小糗 X D 。

我們這次的值星官就是平常負責監督我們打掃的學長，看他撿了短髮、帶上墨鏡、一臉不苟言笑的樣子……看了就想笑（歐），雖然還不是很認識，但就是覺得以他的個性因該會很辛苦，很辛苦的當個不能笑的嚴肅值星官。

雖然外表看起來真的威風挺挺，但從喊話的音量、內容、舉止，還是有一點點可以感覺得出來學長平常並不是那種有嚴肅形象的人，淡淡的散發出壓抑平常自己的痛苦，但也可以感覺到學長是認真的想扮演好值星官的角色。

雖然我很無良的看到他就偷偷的笑，但也為他豎起大拇指 XD 。

第二天上午有個小小的運動會，原本是安排了五個比賽，但因時間的關係只比了四個。

在這運動會裡，我們 ” 小象邦邦隊 ” 威力全開，從第一關到第三關都是高分組，很遺憾的第四關陷在麵粉裡的乒乓球出奇的難吹，不管怎麼 吹，吹的四周都是麵粉飛揚，那乒乓球還是聞風不動，而沒取得高分。最後和另外兩個組同分，前三名。

下午則是大地遊戲，兩小組組成一大組圍成一個圈，然後伸手去牽除了隔壁之外任何組員的手，然後由兩位被挑出來的天才負責解開由雙手連接起來的圓。

我們這組很特別，也不知道是怎牽的，最後解開總是不只一個圈，而是好幾個圓圈互相交錯。

這遊戲以前似乎也曾玩過一次，只是那次沒有像這次組員間那麼配合，最後不了了之。

解玩幾次圓後，再來就是玩水了。

學長姐有事先提醒要多帶套衣服更換，而在看到學長姐們準備的道具……水槍、水球、水桶、水管……根本是打定要全部人都濕透。

在一陣槍林彈雨中，果真無一倖免，就算真有人武功高強的躲過所有攻擊，但最後還是會被全身濕透的傷兵們緊緊抓住，來個集體轟炸。

在最後的最後，有個寫背時間，我原以為”背”是動詞，沒想到是名詞。每人都拿到一張 A4 白紙貼在背上，而想留言給誰就在他背後的紙上寫感言。

這樣的”寫背時間”我還是第一次遇到，很常看到一個人的背後接了長長的人龍，一個接一個的寫，真是很有趣的畫面。幾乎每個人背後都有留言，有些超夯人氣王的流言更是寫到翻面還增紙，看來經過這次迎新大家都熟絡了不少。

回到家後整個累癱，第一次想翹掉星期一的課慾望那麼強烈，衣服也超 ~ 懶的手洗，通通交給洗衣機大嬸了。

而在山上的兩天艷陽直射，雖然有準備防曬，但沒想到有漏塗之肌，之後的幾天的洗澡換衣真是一大痛苦，討厭的曬傷。

之後幾天在前往上課地點的路上，常常會看到一些眼熟的面孔。

資管的學長姐、應日的學長姐、同班同學、應日的同學，雖然不是非常的熟，但都大約認的出來，看來這次迎新真的讓我多認識了些平常不會有交集的人。

很感謝學長姐的用心及努力。
